# Source Code

This folder contains the source code for the IEEE Access manuscript:

"Predicting the Number of Clusters in Genomic Data via Synthetic Pre-training and SVR-Based Ordinal Regression".

The implementation includes:
- k-mer feature extraction
- PCA-based dimensionality reduction
- KDE-based synthetic pre-training data generation
- SVR-based Regressor Chains (SVRRC) model
- Experimental evaluation pipeline
